Om Uppgiften

På den här sidan har jag använt båda ramverken för att stila. Till exempel kan du byta stil på navigeringsmenyn (navbar) mellan Bootstrap och Tailwind. Undersidan för Bootstrap kommer att stylas med enbart Bootstrap, och på samma sätt kommer undersidan för Tailwind att stylas endast med Tailwind CSS.

- uppgiften går på att skapa en webbplats med totalt fyra undersidor (Hem, CSS-ramverk-1, CSS-ramverk-2 och publicering) som använder två olika CSS-ramverk för att styla sidorna.

  • Uppgiften's krav
  • 1- Webbplatsen ska bestå av följande undersidor:
    • Hem: Huvudsidan som ger en överblick över projektet.
    • Bootstrap: En sida som visar exempel på komponenter och styling med Bootstrap.
    • Tailwind: En sida som visar exempel på komponenter och styling med Tailwind CSS.
    • Publicering: En sida som beskriver hur du publicerade webbplatsen och vilket verktyg jag använde.
  • 2- CSS-ramverk:
    • Använd Bootstrap för att bygga och styla en av undersidorna..
    • Använd Tailwind CSS för att bygga och styla en annan undersida.

Bootstrap CSS-ramverk

Bootstrap är ett känt och flexibelt CSS-verktygslåde som låter oss att definiera element (knapparna,cards/formulär,navigationsmenyerna ) som kan användas för att snabbare skapa webbsidor. Det använder ett klasssystem för att tillämpa design och layout.

Tailwind CSS-ramverk

Tailwind CSS är en ramverk CSS för utility-first som gör att vi kan skapa design genom att använda små CSS-klasser direkt i HTML-koden istället för att skapa anpassad CSS-kodning. Istället för att skapa våra egna CSS från grunden använder vi färdiga klasser såsom bg-blue50-, text-center och -p och så vidare.
